	- Поле ввода логина обязательное и должно содержать не менее 4 символов
		input type="text" name="login" required minlength="4"

		Регулярное выражение для проверки пароля - 
		required pattern="(?=.*\d)(?=.*[a-z])(?=.*[A-Z])(?=.*[!@#\$%\^&\*]).{6,}"
