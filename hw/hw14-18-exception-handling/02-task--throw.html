Задание:
  Внесите правки в функцию sum.
  Сделайте так, чтобы при отсутствии одного из параметров, функция выбрасывала ошибку.

Throw - оператор, использующийся для создания исключения,
оператор есть смысл использовать для библиотечных функций, 
которые используются в разных частях программы 
и должны уведомлять разработчика о неправильном использовании.

function sum(a, b) {
  if (!a) throw new Error(`параметр: а, необходимо заполнить!`);
  if (!b) throw new Error(`параметр: b, необходимо заполнить!`);
  
  return a + b;
}

try {
  let result = sum(1, 5);
  console.log(result);

  result = sum(10);
  console.log(result);
} catch (error) {
  console.log(error.message);
}
