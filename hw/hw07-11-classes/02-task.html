Задание:
	Определите класс цилиндр, 
	который будет объектным представлением геометрического тела.
	Класс должен содержать конструктор.
	Конструктор должен принимать два параметра высота и радиус 
	и инициализировать соответствующие свойства.
	В классе должен быть определен метод getVolume, 
	который должен возвращать значение объема данного цилиндра.
	Создайте два экземпляра данного класса, 
	передав произвольные значения в качестве параметров конструктора 
	и выведите на консоль значения объема двух экземпляров.
